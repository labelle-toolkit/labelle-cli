labelle-cli

Command-line interface for labelle-engine projects

Install with one command

curl -fsSL https://labelle-toolkit.github.io/labelle-cli/install.sh | bash
Linux x86_64 Linux ARM64 macOS x86_64 macOS ARM64 Windows x86_64

Version Management

Fetches engine versions from GitHub releases automatically

Project Generation

Delegates to the engine's generator for build file creation

Version Pinning

Projects specify their engine version in project.labelle

Cross-Platform

Pre-built binaries for Linux, macOS, and Windows

Commands

labelle init <name> Create a new labelle project
labelle generate Generate project files from project.labelle
labelle build Build the project
labelle run Build and run the project
labelle update Clear caches and regenerate
labelle upgrade Upgrade to a newer engine version
labelle upgrade --list List available engine versions
